# Design System

Design Varguard Studio like a professional compute infrastructure platform.

Visual inspiration:
- Databricks
- Grafana
- RunPod
- Modal
- Linear
- Retool

But avoid generic startup SaaS aesthetics.

The interface should feel:
- compact
- stable
- operational
- technical
- enterprise-grade

Use:
- dark sidebar
- muted neutral palette
- infrastructure dashboard layout
- dense information hierarchy
- subtle technical visuals

Avoid:
- oversized whitespace
- playful illustrations
- heavy gradients
- glassmorphism
- neon cyberpunk

The product should feel like:
“managed GPU operations center”
instead of
“modern startup dashboard”.

## UI Philosophy

The interface must feel:
- compact
- operational
- infrastructure-oriented

## Colors

Primary accent:
- deep cyan
OR
- electric blue

Background:
- neutral/slate/dark

## Components

Prefer:
- strong cards
- thin borders
- muted panels
- status badges
- compact tables

Avoid:
- excessive whitespace
- oversized hero sections
- glassmorphism
- neon cyberpunk
